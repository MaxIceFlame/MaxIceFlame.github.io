# 冰焰模板 4.6.0
*****
## 简介
早期基于狗带模板V7.2开发的饭制模板，加入多种功能，在圈子中被广泛使用。4.0版本后完全重构，优化了代码逻辑，并加入了更多功能

## 使用要求
### 2.0至3.0模板
Unity版本为2018.3.5及以上

### 4.0模板
Unity版本为2021.3.14及以上
